# Incident Response

`incident-response/` contains scenario and runbook scaffolds for MERIDIAN LABS.

Incident-response content should be based on validated platform behavior. The
current files are seeds for future investigation exercises, not proof that the
underlying detections or response automation exist.

## Current Scope

- Segmentation-violation runbook seed.
- Guest-to-internal-access scenario seed.

## Planned Scope

- Scenario timelines.
- Expected telemetry.
- Investigation steps.
- Containment and recovery notes.
- Lessons learned that feed back into the threat model.
