# Network Security

`network-security/` contains architecture-driven modules for segmentation,
secure connectivity, and threat-prevention controls.

This directory is the target home for platform network contracts. Existing
guided lab content remains in `labs/` for compatibility and will be migrated
later.

## Current Scope

- Structured segmentation flow contract.
- Connectivity module scaffold.
- Mapping from current guided lab intent to future platform validation.

## Planned Scope

- Local topology implementation for allowed and denied paths.
- Evidence for segmentation validation.
- Secure branch-to-HQ connectivity design and tests.
- Threat-prevention controls tied to explicit abuse cases.
