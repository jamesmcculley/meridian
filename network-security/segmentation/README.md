# Segmentation

This module defines network segmentation intent for MERIDIAN LABS as structured
platform data.

`flows.yaml` is based on the original design intent in
`labs/01-network-segmentation-trust-boundaries/flows.md`. The original file is
preserved in place as guided-lab content and historical context.

## Current Scope

- Allowed and denied flow contract.
- Zone definitions from the first Branch A model.
- Assumptions and local-lab limitations.

## Not Implemented Yet

- Firewall rules.
- Docker network topology for these flows.
- Automated reachability tests.
- Evidence files proving allowed and denied paths.
