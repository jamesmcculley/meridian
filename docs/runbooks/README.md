# Runbooks

This directory is reserved for MERIDIAN LABS operational runbooks.

Planned runbooks:

- branch-to-HQ connectivity failure
- unexpected access between guest and employee zones
- telemetry collector ingest or search failure
- local service unavailable from an allowed subnet
- detection rule fires unexpectedly
- policy check failure triage
- Trivy finding triage

Runbooks should be added after the related lab behavior exists. They should focus
on evidence, validation commands, likely causes, and recovery steps.
