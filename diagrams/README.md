# Diagrams

This directory is reserved for MERIDIAN LABS architecture and topology diagrams.

The first real diagram is intentionally left as a hands-on task after the initial
segmentation model is understood. Good candidates:

- distributed office context diagram
- zone and trust-boundary diagram
- allowed and denied flow diagram
- telemetry flow diagram

Use a format that is easy to review in pull requests. Mermaid, draw.io exports,
or checked-in image exports are acceptable when they support the lab.
