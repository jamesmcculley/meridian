# Identity And Access

`identity-access/` is the planned home for users, roles, service identities,
access policies, and audit-event contracts.

This directory is an initial scaffold. Existing Vault configuration under
`onprem/` is preserved as a reference service, but it is not yet a full identity
platform for MERIDIAN LABS.

## Planned Scope

- Role and persona definitions.
- Service identity contracts.
- Admin versus user access boundaries.
- Access policy examples.
- Audit events that feed telemetry and detection scenarios.

## Not Implemented Yet

- IdP integration.
- Real user lifecycle management.
- Production-grade secret distribution.
