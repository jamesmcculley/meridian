# Lab 03: Host Hardening

This lab establishes baseline controls for lab hosts and service placeholders.

## Objective

Define and apply a small hardening baseline that supports the distributed office
model without pretending the lab is a production endpoint management platform.

## Current State

Scaffold only. The first real hardening playbook is intentionally left as a
hands-on learning task.

## Expected Evidence

When implemented, this lab should show:

- baseline service exposure decisions
- least-privilege administrative access assumptions
- simple configuration checks
- repeatable validation steps
- clear notes about local lab limitations

See `LEARNING_TASK.md` for the implementation prompt.
